# Getting Started

A quick guide on getting started using SwiftMIDISync.

![swift-midi-sync](swift-midi-sync-banner.png)

## 1. Import the Library

Add the SwiftMIDISync module to your project, and import it.

```swift
import SwiftMIDISync
```

## 2. Example Code

See the **MTC Example** example project in the Examples folder for a demonstration of ``MTCGenerator`` and ``MTCReceiver``.
